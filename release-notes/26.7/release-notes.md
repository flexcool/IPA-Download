## 简体中文

感谢你使用 Pastel！

我们很高兴带来 Pastel 26.7。

此次更新内容为：

1. 支持搜索时区分 iPhone、iPad 和 Apple Vision Pro App，并新增了对 Apple Vision Pro 的支持。
2. “手动获取”选项现在可在下载页面提供，并支持仅输入 App ID 获取最新版。
3. App ID、版本 ID 支持手动拷贝。
4. 大量细节优化和用户体验提升。

即将到来：Windows 版本适配。

Pastel 26.6 更新内容：

1. 修复部份 UI 显示效果。
2. 支持多选 App 版本进行下载、删除。
3. 从 Apple 获取来源时检测到从未获取的 App，会进行二次确认。

## English

Thank you for using Pastel!

We are happy to bring you Pastel 26.7.

What's new:

1. Search can now distinguish between iPhone, iPad, and Apple Vision Pro apps, with new support for Apple Vision Pro.
2. Manual fetch is now available on the Downloads page and can fetch the latest version with only an App ID.
3. App IDs and version IDs can now be copied manually.
4. Many detail refinements and user experience improvements.

Coming soon: Windows version support.

Pastel 26.6 changes:

1. Fixed several UI display issues.
2. Added support for selecting multiple app versions to download or delete.
3. When using Apple as the source, Pastel now asks for confirmation if the app has never been acquired before.

## ภาษาไทย

ขอบคุณที่ใช้ Pastel!

เรายินดีที่ได้นำ Pastel 26.7 มาให้ใช้งาน

สิ่งที่อัปเดตในเวอร์ชันนี้:

1. รองรับการแยกผลการค้นหาระหว่าง iPhone, iPad และ Apple Vision Pro App พร้อมเพิ่มการรองรับ Apple Vision Pro
2. ตัวเลือกดึงข้อมูลด้วยตนเองพร้อมใช้งานในหน้าดาวน์โหลดแล้ว และรองรับการดึงเวอร์ชันล่าสุดด้วยการใส่เฉพาะ App ID
3. รองรับการคัดลอก App ID และ Version ID ด้วยตนเอง
4. ปรับปรุงรายละเอียดจำนวนมากและยกระดับประสบการณ์การใช้งาน

เร็ว ๆ นี้: การรองรับเวอร์ชัน Windows

การเปลี่ยนแปลงใน Pastel 26.6:

1. แก้ไขการแสดงผล UI บางส่วน
2. รองรับการเลือก App หลายเวอร์ชันเพื่อดาวน์โหลดหรือลบ
3. เมื่อใช้ Apple เป็นแหล่งข้อมูล หากตรวจพบว่าเป็น App ที่ยังไม่เคยได้รับมาก่อน Pastel จะขอการยืนยันอีกครั้ง
