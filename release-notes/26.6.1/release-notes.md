## 简体中文

感谢你使用 Pastel！

我们很高兴带来 Pastel 26.6.1。

此次更新内容为：

1. 修复部份 UI 显示效果。
2. 支持多选 App 版本进行下载（按 Command 即可）。
3. 从 Apple 获取来源时检测到从未获取的 App，会进行二次确认。

## 繁體中文

感謝你使用 Pastel！

我們很高興帶來 Pastel 26.6.1。

此次更新內容為：

1. 修復部分 UI 顯示效果。
2. 支援多選 App 版本進行下載（按 Command 即可）。
3. 從 Apple 取得來源時偵測到從未取得的 App，會進行二次確認。

## English

Thank you for using Pastel!

We are happy to bring you Pastel 26.6.1.

What's new:

1. Fixed several UI display issues.
2. Added support for selecting multiple app versions to download by holding Command.
3. When using Apple as the source, Pastel now asks for confirmation if the app has never been acquired before.
